# BIO_MODELS

BIO_MODELS is a web-based collection of biological figures from the Ethiopian Grades 9–12 Biology curriculum, organized to provide clear and accessible visual reference for students.

---

##  Overview

This project brings together key biological diagrams and figures from the Ethiopian high school biology textbooks into a single, structured platform.
It is designed to make studying more efficient by allowing students to quickly find and review important visuals.

---

##  Features

* Organized collection of biology figures
* Clean and simple user interface
* Easy navigation by topic or category
* Focused on Grades 9–12 Ethiopian Biology curriculum

---

##  Purpose

The goal of this project is to:

* Improve access to important biology figures
* Support students in understanding visual concepts
* Provide a centralized reference for study and revision

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/MGM50/BIO_MODELS.git
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```
